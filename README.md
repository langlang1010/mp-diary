# mp-v-diary
微信小程序；微V日记前端代码。

### 编写工具

微信公众平台提供的开发工具。

### 代码简介

* 使用 `iView` UI框架开发，参考[官网地址](https://weapp.iviewui.com/docs/guide/start)

* 使用 `iconfont` 图标库，参考[官网地址](https://www.iconfont.cn/)
* 参考[微信小程序官网实例](https://github.com/wechat-miniprogram/miniprogram-demo)

* 循环查文档 写代码 查文档
